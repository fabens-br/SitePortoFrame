# DIRETRIZ PERMANENTE — PADRONIZAÇÃO FOTOGRÁFICA PREMIUM

A partir deste momento, considere que TODAS as imagens do site deverão seguir um único padrão fotográfico institucional.

========================================================

## OBJETIVO

O visitante deve ter a sensação de que todas as fotografias foram produzidas pela mesma equipe de arquitetura. Nenhuma imagem deverá parecer retirada de bancos de imagens diferentes.

========================================================

## PADRÃO VISUAL

Todas as imagens deverão possuir automaticamente:
• mesma temperatura de cor;
• mesma luminosidade;
• mesmo contraste;
• mesma saturação;
• mesma nitidez;
• mesmo enquadramento;
• mesma linguagem arquitetônica;
• mesma sensação premium.

O resultado deve transmitir:
• engenharia;
• arquitetura;
• alto padrão;
• tecnologia;
• conforto;
• sustentabilidade.

========================================================

## TRATAMENTO

Sempre que uma nova imagem for adicionada ao projeto:
aplique automaticamente o mesmo tratamento visual (filtros CSS de temperatura, brilho, contraste, saturação) utilizado nas demais. Nenhuma imagem poderá destoar do restante do site. Esta padronização deverá valer para Hero, Projetos, Mitos e Verdades, Financiamento, CTA e demais seções futuras.
